# Docker Alpine image for PHP FPM and CLI for Symfony

- php-fpm 7.x
- php-mysql + mysql tools
- php opcache + iconv + exif + gd + intl + xsl + xml + json + zip
- composer
- nodejs + npm + yarn
- dev tools (bash, nano, curl, unzip, git, rsync)
